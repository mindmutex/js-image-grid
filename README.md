# Simple Image Grid

Simple jQuery plugin for displaying images in a tiled grid.

## Getting Started

```html
<script src="jquery.js"></script>
<script src="dist/image-grid.min.js"></script>
<script>
jQuery(function($) {
  $('.image-grid').imageGrid({
  });
});
</script>
```
